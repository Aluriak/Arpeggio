Arpeggio - PEG parser
=====================

|pypi-badge| |license| |build-status| |docs|

Arpeggio is a recursive descent parser with memoization based on PEG grammars
(aka Packrat parser).

Documentation with tutorials is available `here
<http://igordejanovic.net/Arpeggio/>`_.
